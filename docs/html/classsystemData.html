void systemData::updateConstraints ( double  time)

Updates all relevant rigid body motion tensors, respective gradients, and kinematic/Udwadia constraints.

Many functions are called and there is a dependency chain between them. All function calls can be run in any order besides gradientChangeOfVariableTensors(), which depends on rigidBodyMotionTensors() being run first. This is assuming the Udwadia linear constraint system ( \( \mathbf{A} \, \ddot{\boldsymbol{\xi}} = \mathbf{b} \) ) is independent of configuration.

Parameters
time(dimensionless) simulation time to update parameters.
