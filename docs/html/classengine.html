void engine::run ( )

Runs the simulation from \( t_0 \) to \( t_0f \).

Method creates an Eigen::ThreadPool and Eigen::ThreadPoolDevice that is passed to the integrate() method to speed up Eigen::Tensor computations. Method also calculates the total number of integration steps required and manages the output of the progressBar class.

< number of threads in pool for eigen calculations (set to number of physical cores on machine)

< thread pool device with access to all threads
